成績から見えること

国語力が最大の武器

  • 偏差値59.9は日能研生の中でも上位層。読書・作文好きと完全に一致
  • 2科（国算）偏差値56.1 と 4科偏差値51.5 の差が大きい → 国語が全体を引き上げている
  • この国語力は「適性検査型」入試で最大限に活きる（長文読解＋記述＋作文が中心）

理科 — 好きだけど偏差値はまだこれから

  • 公開模試の偏差値は46.2だが、春期講習テストでは評価8（最高水準）
  • 興味はあるが、テスト形式の知識問題ではまだ点が取り切れていない
  • 理科は小5〜6で内容が本格化するため、今の数字で判断するのは早い
  • → 好きという気持ちがある子は、環境さえ整えば伸びる

算数が負担の中心になっている

  • 偏差値51.1は「できない」わけではないが、本人が嫌い → 塾のストレスの大部分がここ
  • 私立4科受験は算数の比重が非常に大きい（配点150点の学校も多い）
  • 算数に苦しみながら受験勉強を続けると、理科や読書の好奇心まで潰れるリスク
  • → 算数の負荷が少ない受験方式を選ぶのが湊くんには合う

3つの進路 — 湊くんに合うのはどれ？

A. 公立中高一貫校湊くんの強みが最も活きる

適性検査＝国語力で戦える 学費は公立と同じ 6年一貫で理科を深められる 倍率が高い（4〜5倍） 落ちたら地元公立中

なぜ湊くんに合うか：

  • 適性検査は「長い文章を読んで考えを書く」問題 → 国語偏差値60、作文好きが直接活きる
  • 算数も出るが、計算スピードより「考え方を説明する」形式 → 国語力でカバーできる
  • 4科の暗記型テストではなく、教科横断の思考問題 → テスト地獄からの解放
  • 不合格でも地元公立中に進めるので、ルートCにスライドできる

塾について：適性検査対策は私立受験塾とは内容が違う。日能研を続けるか、適性検査専門塾に切り替えるか要検討。私立型より負荷は軽い場合が多い。

B. 私立中学

理科設備が充実 学校の選択肢が多い 学費が高い（年80〜120万円） 算数の比重が大きい入試が多い

湊くんの場合：

  • 4科受験だと算数がネックになりやすい → 2科（国算）受験可能な学校を選ぶと有利
  • 偏差値56（2科）で狙える中堅校に良い学校がある
  • 大学附属校なら中学入学後は受験プレッシャーなしで、理科をじっくり学べる
  • ただし小5〜6の塾の負荷は今より確実に増える。算数嫌いの湊くんには覚悟が要る

C. 地元の公立中学 → 高校受験

受験ストレスから完全解放 費用最小 友達と一緒に進学 高校受験は必要 理科の環境は学校次第

湊くんの場合：

  • 塾のストレスから解放され、小学校生活を楽しめる
  • 中学で科学部に入れば、理科への興味は十分に深められる
  • 国語力がある子は高校受験にも強い（内申点＋記述力）
  • 高校で多摩高校（SSH）や県立の理数系高校を目指すルートがある（YSFH高校も高校受験なら受験可能な場合あり ※要確認）
  • → 「今は受験しない」は「理科を諦める」ではない

判断のヒント

湊くんに一番おすすめできるルートは？

成績データと性格を総合すると：

「川崎市立川崎高附属中を第一志望に、適性検査対策に切り替える」

  • 川崎市在住なので受験資格がある唯一の公立中高一貫校
  • 国語偏差値60＋作文好き → 適性検査で最も力を発揮できる
  • 算数地獄から解放される（適性検査の算数は記述型で、暗記・スピード勝負ではない）
  • 探究学習で理科を6年間じっくり学べる
  • 学費が公立と同じ。きょうだいの教育費も考えると経済的
  • 自宅から近い（約15〜20分）ので通学の負担も小さい
  • 落ちても地元公立中に行くだけ → リスクが小さい

注意点：倍率4〜5倍なので「受かったらラッキー」くらいの気持ちで。落ちた場合のルートC（公立中→高校受験）も、湊くんには十分良い選択肢。

「塾がつらい」への向き合い方

  • テストが多い → 日能研の4科テスト量が合っていない可能性。適性検査対策に切り替えれば、テストの質が変わる
  • 算数が嫌い → 私立4科型を続ける限り、算数の負荷は増え続ける。適性検査型なら算数は「考えを書く」スタイルで、湊くんの国語力でアプローチできる
  • 受験の目的がわからない → 「この学校に行きたい」が見つかれば変わる。川崎高附属中の体験授業に連れて行くと、本人の反応で判断できる
  • 「受験をやめる＝負け」ではない → 公立中から高校でSSH校に進むルートでも、理科を極められる道はちゃんとある

親として考えておきたいこと

  • 真面目だが忍耐力は控えめ → 面倒見のよい学校の方が安心。放任型の進学校よりサポートがある環境が合う
  • きょうだいの受験も未定 → 二人分の教育費を見据えると、公立中高一貫は経済的なメリットが大きい
  • 小4の今なら方向転換の時間は十分ある。小5の夏までに方向性を決められれば問題なし
  • 学校説明会・文化祭で本人が「ここに行きたい」と思えるかが最終判断の鍵

湊くんに読んでほしいページ

ここから先は、湊くん自身に読んでもらうために書いた内容です。

湊くん、理科が好きなんだよね？

実験とか、化学とか、物理のしくみを調べるのが好き。それってすごく大事な気持ちだよ。
中学校には、その「好き」をもっと深められる場所がある。学校によって、どれだけ理科をやれるかが全然ちがうんだ。

中学校には3つの道がある

1. 探究型の中学校に挑戦する
「川崎市立川崎高附属中」っていう学校がある。ここは6年間通えて、「探究」っていう、自分でテーマを決めて調べたり実験したりする特別な授業がある。入るにはテストがあるけど、算数の計算スピードを競うタイプじゃなくて、文章を読んで、自分の考えを書くテスト。国語が得意な湊くんには向いてると思う。家からも近いよ。

2. 私立の中学校に行く
理科の設備が充実した私立中学もある。ただ、入試は算数がけっこう大変になる。

3. 近くの中学校にそのまま行く
今の友達とそのまま一緒に中学に行く道。受験勉強はしなくていい。中学校で理科部に入って実験できるし、高校から理科の専門コースに行くこともできる。

どの道でも、理科はできる

大事なのは、「理科をもっとやりたい」っていう気持ち。それさえあれば、どの道を選んでも理科を続けられる。

塾がつらいと感じてるなら、それはちゃんと伝えていいこと。つらいのをガマンし続けるより、自分に合ったやり方を見つけるほうが大事だから。

まずは一回、川崎高附属中を見に行ってみない？ 体験授業があるかもしれない。行ってみて「ここいいな」と思ったら、それが一番のヒント。「別にいいや」と思ったら、それも大事な答え。

よくある疑問

適性検査の対策は、今の日能研でできる？
日能研にも公立中高一貫対策コースがある校舎もあります。ただ、私立向けカリキュラムがベースなので、適性検査専門塾（ena、栄光の公立一貫コースなど）への切替も選択肢。塾の先生に相談してみてください。
途中で塾をやめたら、今までの勉強は無駄になる？
無駄にはなりません。日能研で身につけた読解力・思考力・学習習慣は、適性検査対策でも高校受験でも大きなアドバンテージになります。
公立中から理系の道に進むのは不利？
不利ではありません。高校のSSH校（スーパーサイエンスハイスクール）は公立中からも十分目指せます。大学受験で最終的に並ぶケースも多いです。
川崎高附属中に落ちたらどうなる？
地元の公立中学にそのまま進学します。公立中高一貫の不合格は「ルートCに進むだけ」で、マイナスはありません。高校受験で理数系の高校を目指すことができます。
算数が嫌いでも受験できる？
適性検査型の入試なら、算数は「計算の速さ」ではなく「考え方を文章で説明する」形式。国語力の高い湊くんなら、むしろ得意な形で算数に取り組める可能性があります。
本人にやる気がないのに受験させていいの？
「塾が嫌い」と「学びたくない」は別の話です。塾の形式が合っていないだけかもしれません。「この学校で理科をやりたい」という動機が見つかれば、状況は変わります。まずは学校見学で本人の反応を見ることをおすすめします。
